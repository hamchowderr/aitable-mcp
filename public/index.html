This is a Model Context Protocol (MCP) server providing programmatic access to AITable datasheets, fields, records, and more.

MCP Protocol Endpoints

Streamable HTTP Transport (Primary):
/mcp
SSE Transport (Alternative):
/sse

Using with Claude Desktop

Add this server to your Claude Desktop configuration:

{
  "mcpServers": {
    "aitable": {
      "url": "https://your-deployment-url.vercel.app/mcp"
    }
  }
}

Available Tools

This server provides 16 AITable tools organized into categories:

Database Tools

  • get_records - Query datasheet records with filtering, sorting, pagination
  • create_records - Create new records (max 10 per request)
  • update_records - Update existing records (max 10 per request)
  • delete_records - Delete records (max 10 per request)
  • get_fields - Get field metadata from a datasheet
  • create_field - Create a new field in a datasheet
  • delete_field - Delete a field from a datasheet
  • get_views - Get all views from a datasheet
  • create_datasheet - Create a new datasheet
  • upload_attachment - Upload files to attach to records

Node Tools

  • get_node_list - List nodes in the space working directory
  • get_node_detail - Get detailed information about a node
  • search_nodes - Search for nodes by type and permissions
  • create_embed_link - Create embeddable links for nodes
  • get_embed_links - List all embed links for a node
  • delete_embed_link - Delete/disable an embed link
